== Description ==

Image gallery for WordPress with classic crossfade effect

== Installation ==

1. Upload crossfade gallery folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php if(function_exists('crossfade_gallery')) crossfade_gallery(); ?>` in your templates
